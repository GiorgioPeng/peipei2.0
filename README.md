# PeiPei2.0
这是宿舍匹配系统的第二个版本，美化了UI界面，基于统计的学生信息来给学生分配宿舍

# 使用该系统可能导致的问题
1. 同一宿舍的舍友之间可能各方面难以达成互补、多彩，而是性格、习惯等方面相似，可能难以促进人的全面发展及增加见识的广度，更甚者会增加或强化恶习

# 结果
1. 经过两年在北京交通大学威海校区试用，从统计上来看增加了宿舍的和谐性，提出换寝的学生数量大大减少
